# Star Fit

Sistema web para gerenciamento e acompanhamento de atletas, professores, nutricionistas e instituições esportivas.

O objetivo do projeto é centralizar informações relacionadas a treinos, avaliações físicas, metas e desempenho em uma única plataforma, promovendo uma gestão mais eficiente e acessível.

---

## 📖 Sobre o Projeto

O **Star Fit** foi idealizado para resolver problemas comuns no acompanhamento esportivo, como:

- Informações dispersas em diferentes plataformas;
- Dificuldade no acompanhamento da evolução dos atletas;
- Falta de comunicação entre professores, atletas e nutricionistas;
- Ausência de métricas centralizadas para tomada de decisão.

A plataforma permitirá que diferentes perfis de usuários interajam conforme suas necessidades, garantindo praticidade, organização e segurança.

---

## 🎯 Objetivos

- Centralizar dados esportivos em um único ambiente;
- Facilitar o acompanhamento da evolução dos atletas;
- Permitir a criação de treinos personalizados;
- Disponibilizar relatórios e gráficos de desempenho;
- Incentivar o engajamento por meio de metas e rankings;
- Melhorar a comunicação entre atletas e profissionais.

---

## 👥 Usuários do Sistema

### 🏃 Atleta

- Visualizar treinos e atividades;
- Acompanhar metas pessoais;
- Consultar avaliações físicas;
- Visualizar gráficos de desempenho;
- Participar de rankings.

### 👨‍🏫 Professor

- Cadastrar e gerenciar atletas;
- Criar treinos personalizados;
- Registrar avaliações físicas;
- Definir atividades e metas;
- Monitorar o progresso dos atletas.

### 🥗 Nutricionista

- Acessar relatórios autorizados;
- Consultar indicadores físicos;
- Acompanhar a evolução dos atletas;
- Auxiliar na definição de estratégias nutricionais.

### 🛠️ Administrador

- Gerenciar usuários;
- Controlar permissões de acesso;
- Monitorar o funcionamento da plataforma;
- Gerenciar instituições cadastradas.

---

## ✨ Funcionalidades

- Cadastro e autenticação de usuários;
- Controle de permissões por perfil;
- Gerenciamento de atletas;
- Criação de treinos adaptados;
- Registro de avaliações físicas;
- Acompanhamento de metas;
- Geração de relatórios;
- Visualização de gráficos de desempenho;
- Sistema de rankings;
- Centralização de dados esportivos.

- Como atleta, quero visualizar meus resultados para acompanhar minha evolução.
- Como atleta, quero participar de rankings para comparar meu desempenho.
- Como professor, quero cadastrar treinos personalizados para meus alunos.
- Como professor, quero definir metas e atividades para casa.
- Como nutricionista, quero acessar relatórios dos atletas.
- Como administrador, quero gerenciar usuários e permissões.
