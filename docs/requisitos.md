# Requisitos do Sistema

## Requisitos Funcionais

| Código | Descrição |
|---------|-----------|
| RF01 | Cadastrar usuários |
| RF02 | Autenticar usuários |
| RF03 | Gerenciar atletas |
| RF04 | Registrar avaliações físicas |
| RF05 | Criar treinos personalizados |
| RF06 | Gerenciar metas e atividades |
| RF07 | Gerar relatórios |
| RF08 | Exibir gráficos de desempenho |
| RF09 | Gerar rankings |
| RF10 | Controlar permissões de acesso |

---

## Requisitos Não Funcionais

| Código | Descrição |
|---------|-----------|
| RNF01 | O sistema deve ser responsivo |
| RNF02 | O tempo de resposta deve ser inferior a 3 segundos |
| RNF03 | Os dados devem ser armazenados com segurança |
| RNF04 | O sistema deve funcionar nos principais navegadores |
| RNF05 | O acesso deve exigir autenticação |
| RNF06 | O sistema deve realizar backups periódicos |
| RNF07 | O sistema deve estar disponível 24 horas por dia |
